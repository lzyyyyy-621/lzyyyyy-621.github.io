快速建站小结

主要核心参考了知乎上这篇 https://zhuanlan.zhihu.com/p/35668237

其成品的效果图：https://godweiyang.com/

中间bugs

  • node版本过低，自己去下载新的
  • 自己把github账号记错了 😂
  • github添加密钥 （本身是可以的，拓展可以参考https://www.cnblogs.com/yuqiliu/p/12551258.html）
  • 上传
    • 用hexo d不成功
    • 更新方法见 https://ismartinmeng.com/2020/05/25/Deploy-you-personal-blog-in-10-min-with-Hexo-AWS-Route53-GithubPage-05-2020/
      • 民工三连提交：git init git add . git commit -m "init"
    • 总结一下以后的流程是：
      • 在typora上写，保存到source文件夹
      • 修改下title【当然这两步可以改成先git新建：hexo new post "article title"，再修改这个.md】
      • 写好之后hexo g, hexo s, hexo d，再民工三连上传！
